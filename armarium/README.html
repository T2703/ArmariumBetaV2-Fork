How to Run Armarium Locally

To run Armarium locally, follow these steps:

  1. Navigate to the Armarium folder: cd armarium
  2. Start the server by running: npm start

How to Add to Wardrobe Library in Armarium Firestore

  1. Navigate to the Firestore Database tab.
  2. Click on the DefaultItemCollection.
  3. Click on the "Add Collection" button.
  4. Enter the name of the clothing item collection you want to add.
  5. Click on the "Add Document" button and start a collection called Items.
  6. Note: Add the image of the clothing item to Firebase Storage first!
  7. Click on the "Add Document" button and add the following fields:
    • color - type: string
    • createdAt - type: timestamp
    • tags - type: array
    • url - type: string (URL of the clothing image in Firebase Storage)
    • title - type: string

Storage

Objects are saved both in Firebase Storage and Firestore. Most items can be queried through Firestore, but larger objects like Outfits and Styleboards are also saved in Firebase Storage.
