How to Run Armarium Locally

To run Armarium locally, you need follow these steps:

First go into the armarium folder `cd armarium`

Then run the following command to start the server: `npm start`

How to Add To Wardrobe Library in Armarium Firestore

Navigate to the Firestore Database tab.

Click on the `DefaultItemCollection`.

Click on the "Add Collection" button.

Enter the name of the clothing item collection you want to add.

Click on the "Add Document" button and start a collection called "Items".

PS: Don't forget to add the image of the clothing item to the firebase storage first!

Click on the "Add Document" button and add these fields:

`color` - type: string

`createdAt` - type: timestamp

`tags` - type: array

`url` - type: string [url of the clothing image in firebase storage]

`title` - type: string

Storage

Objects are saved both in Firebase Storage and Firestore, most things are able to be

queried through Firestore, but larger objects like Outfits and Styleboards are also saved in storage.
